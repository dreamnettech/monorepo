CLI utility for [@dreamnet/deploy](https://www.npmjs.com/package/@dreamnet/deploy)

## Usage

```sh-session
$ yarn global add @dreamnet/deploy-cli
$ deploy ./build/myapp.exe -p ipfs -p pinata -p mega
$ deploy --help
```
